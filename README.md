# Luminé Salon Studio Website

A modern, elegant, and responsive **salon website design showcase** created using **HTML, CSS, and JavaScript**.

This project was designed and developed by **Kanishka Deshan** as part of the creative portfolio work of **CK IT OPS**.

---

## About This Project

**Luminé Salon Studio** is a demo one-page website concept designed for a premium beauty salon.  
The design focuses on a stylish, luxury-inspired user experience with sections for:

- Hero introduction
- About the salon
- Services
- Gallery showcase
- Beauty packages
- Stylist profiles
- Testimonials
- Booking/contact section
- Custom loading screen

This website is created only for **design showcase purposes** using simple front-end technologies.
